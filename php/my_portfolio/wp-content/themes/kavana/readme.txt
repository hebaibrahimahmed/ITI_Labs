== Description ==

Kavana is a minimal design style for WordPress full site editing and is fully compatible with the Gutenverse plugin. This Template is perfect for creators, bloggers, writers, and creatives to suit lifestyle bloggers & magazines. The templates are fully responsive and look great on any device. We want to make your experience using WordPress full site editor more convenient.
